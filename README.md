# 📞 Contact Form - Java Full Stack Internship Task 2

A responsive Contact Form application developed as part of my Java Full Stack Internship.

---

## 🚀 Project Overview

This project allows users to submit their contact details through a responsive frontend. The submitted information is sent to a Spring Boot backend using REST APIs and stored in a MySQL database.

---

## ✨ Features

- Responsive Contact Form
- Modern UI Design
- Spring Boot REST API
- MySQL Database Integration
- Form Validation
- Store Contact Details
- Clean Folder Structure
- MVC Architecture

---

## ⚙️ How to Run

### Backend

1. Open backend project
2. Configure MySQL
3. Run Spring Boot
4. Backend runs on:

```
http://localhost:8080
```

---

### Frontend

Open

```
frontend/index.html
```

in your browser.
